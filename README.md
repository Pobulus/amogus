## Compiling
It only worked with clang++ which is kinda sus.
- client:
`clang++ -lncursesw amogus.cpp main.cpp -o amogus`
- server:
`clang++ amogus-server-main.cpp -o server`

## Usage 
First start up the server:
`./server`
Then connect your client like so:
`./amogus -i <ip address>`
it uses port 8888
